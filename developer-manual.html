← 回 Developer Workspace

1. AIAKOS 是什麼？

AIAKOS 是 AI農業教育平台的共用核心。所有平台，例如 AI農業氣象教練、AI植物診療師、AI智慧農業決策中心、AI智慧養殖與 AI農業經營大富翁，都應共用 Engine、Service 與 JSON Data。

2. 核心架構

AIAKOS Core
├── engine/
│   ├── WeatherEngine
│   ├── WeatherFusionEngine
│   ├── CropEngine
│   ├── DiseaseEngine
│   ├── DecisionEngine
│   └── PromptEngine
├── services/
│   ├── WeatherAPI
│   └── StationService
├── modules/
└── data/

3. JSON 標準

所有資料必須版本化、可擴充、可追溯來源。核心資料包含 crops.json、diseases.json、pests.json、weather_rules.json、stations.json。

4. Engine 開發規範

Engine 不直接操作 DOM，不依賴特定 UI。每個 Engine 只做單一職責，輸入 JSON，輸出結構化結果。

5. Service 規範

Service 負責資料讀取、API 串接、外部資料清洗。Engine 不直接呼叫外部網站。

6. AI Developer Assistants

  • 🌿 Disease Database Assistant
  • 🐛 Pest Database Assistant
  • 🌾 Crop Database Assistant
  • 🌤 Weather Rule Assistant
  • 🧠 AIAKOS Core Developer Assistant

7. 版本管理

小幅修正使用 v6.1 → v6.2；架構升級使用 v7.0；新增跨平台核心能力使用 major version。

8. 專業平台接入原則

AI花藝設計師學院、AI農業經營大富翁、AI智慧養殖等新平台，應先定義資料需求，再接入 AIAKOS Core，不另建孤立邏輯。
